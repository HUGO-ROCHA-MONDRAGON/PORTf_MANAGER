# Plateforme de Pricing d'Options Financières

Application WPF pour le pricing d'options financières avec données de marché réelles (API Twelve Data).

## Commandes

### Build du projet
```powershell
dotnet build
```

### Lancer les tests
```powershell
dotnet test
```

### Lancer l'application WPF
```powershell
cd Pricing.WPF
dotnet run
```

## Obtenir une clé API

1. Aller sur https://twelvedata.com/pricing
2. Cliquer sur "Sign Up" pour le plan "Basic Free"
3. Créer un compte
4. Copier la clé API fournie (800 requêtes/jour gratuites)
5. Entrer la clé dans l'application au démarrage
